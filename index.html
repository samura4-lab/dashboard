Supabase 실시간 공유 연동 · 데이터 관리
Supabase 실시간 공유 설정 미연동
무료 · 5분 설정 · URL만 공유하면 팀원 모두 실시간 공동 편집

① Supabase 프로젝트 만들기
supabase.com → Start for free → GitHub로 로그인 → New project

② API 키 복사
프로젝트 → Settings → API → Project URL과 anon / public 키 복사

③ 테이블 1개 만들기 (SQL Editor에서 실행)
create table if not exists importos (
  id text primary key default 'main',
  data jsonb,
  updated_at timestamptz default now()
);
alter table importos enable row level security;
create policy "all" on importos for all using (true) with check (true);
위 SQL을 복사해서 Supabase SQL Editor에 붙여넣고 Run 하세요.
팀 공유 URL
아래 URL을 카카오톡·슬랙으로 팀원에게 공유하세요.
URL을 열면 Supabase 설정이 자동 적용되어 바로 같은 데이터에 접속됩니다.
